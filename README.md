Приложение разделено на 3 модуля - Calculator, Dialog и Core, Calculator сам калькулятор, Dialog - окно с ошибками и Core это модуль с интерфейсами других модулей, 
модуль Core нужен для того что бы другие модули могли взаимодействовать друг с другом через общие интерфейсы, например Calculator мог вызвать окно с ошибкой. Сами же модули Calculator и 
Dialog друг о друге не знают. Такая система позволяет легко переиспользовать модули в других проектах или подменять реализацию уже существующих модулей с минимальными усилиями.
Модули  Calculator и Dialog реализовны в рамках MVP архитектуры, которая разделяюет код на 3 логических слоя View - графический визуал и инпут, Model - данные и работу с ними и 
Presenter - код связывающий View и Model, и управляющий ими.
